# number_check
【手写数字识别】Python+深度学习+TensorFlow+算法模型训练+CNN卷积神经网络算法

## 一、介绍
手写数字识别系统，使用Python作为主要开发语言，基于深度学习TensorFlow框架，搭建卷积神经网络算法。并通过对数据集进行训练，最后得到一个识别精度较高的模型。并基于Flask框架，开发网页端操作平台，实现用户上传一张图片识别其名称。
